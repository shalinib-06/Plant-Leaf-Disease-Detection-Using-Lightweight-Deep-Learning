# 🌱 Cross-Crop Plant Disease Detection using MobileNetV2

## 📖 Overview
This project focuses on detecting plant leaf diseases using a lightweight deep learning model (MobileNetV2) and evaluating cross-crop generalization.

## 🚀 Features
- Lightweight MobileNetV2 model
- High accuracy (96.18%)
- Cross-crop generalization (71.25%)
- Comparison with Custom CNN
- Efficient (only 11 MB model size)

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## 📊 Results
| Model | Accuracy | Cross-Crop Accuracy | Model Size |
|------|--------|--------------------|-----------|
| MobileNetV2 | 96.18% | 71.25% | 11 MB |
| Custom CNN | 92.06% | 61% | 127 MB |
